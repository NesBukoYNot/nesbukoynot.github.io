# 🥥 NesBukoYNot

A modern e-commerce storefront for **Advanced Web Design Finals** at **FEU Tech**.

## 👥 The Team
- **Vince Gio Acedillo** (Project Leader)
- **Jandrei Labarento**
- **Stephen Dave Blanza**

## ✨ Key Features
- **Storefront**: Dynamic home page and product catalog.
- **Cart & Checkout**: Real-time management and multi-step process.
- **Accounts**: Secure login, registration, and user profiles.
- **Responsive**: Built for all devices.

## 🛠️ Tech Stack
- **React 19** + **Vite**
- **Tailwind CSS**
- **React Router 7**
- **Lucide Icons**

## 🚀 Quick Start
```bash
# Install
npm install

# Run (Dev)
npm run dev

# Build
npm run build
```
